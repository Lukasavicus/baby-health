7. backend/main.py (MODIFIED)
   Location: /backend/main.py
   Changes:
   - Added StaticFiles import for serving frontend
   - Added Path and os imports for file handling
   - Added static file mounting at root (/)
   - Mounted AFTER all API routes so /api/* routes take priority
   - Updated PORT configuration to read from environment variable
   - Changed default port from 8000 to 8080 via env var
   - Added html=True to StaticFiles for SPA routing support

   Key additions:
   ```python
   from fastapi.staticfiles import StaticFiles
   from pathlib import Path
   import os
   
   # Mount static files (after API routes)
   static_dir = Path(__file__).parent.parent / "frontend" / "dist"
   if static_dir.exists():
       app.mount("/", StaticFiles(directory=static_dir, html=True), name="static")
   
   # PORT from environment
   port = int(os.getenv("PORT", "8000"))
   ```
