## User Story

As an engineer on-call, I want clear env examples and a feature-flag guide so that I can run the stack locally and adjust flags without reading the whole codebase.

## Acceptance Criteria

- [`docs/feature-flags.md`](../feature-flags.md) descreve variáveis, fluxo e segurança (TOTP)
- `LOCAL_RUN.md` e `.env.example` (raiz, backend, app-design, legacy) refletem host `127.0.0.1` e porta `8080` onde aplicável
- Scripts de deploy (`deploy/*.sh`) e notas de delivery alinham proxy para a porta da API
- Nenhum segredo real é versionado (apenas `.env.example`)
