## User Story

Como sistema, eu quero que rotas protegidas rejeitem requisições sem token válido.

## Acceptance Criteria

- All API routes require JWT except /api/auth/login, /health, and /docs
- Invalid or expired tokens return HTTP 401
- Repository is instantiated with profile_dir from token
- No global DATA_DIR env var needed anymore
