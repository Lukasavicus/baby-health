Design a premium hero dashboard tile for a mobile-first baby health app called “Baby Health”.

This tile will be the BABY EQUIVALENT of the Samsung Health core tile / daily activity core.
It should be the main high-level wellness tile on the “Today” screen.

GOAL
Create a single “Baby Core” tile that summarizes whether the baby is doing well and appears healthy today, using a composite score made from the 4 most important baby indicators.

This tile should answer:
“Is the baby doing well today?”
in a way that feels premium, calm, intelligent, and easy to scan.

IMPORTANT
Use Samsung Health as the UX and product philosophy benchmark:
- premium summary tile
- visually central “core” representation
- combined health logic in one elegant card
- highly scannable
- dashboard-first
- confidence-building
- not cluttered

Do NOT copy Samsung branding or exact assets.
Create an original tile that feels like:
“This is the Samsung Health core, but redesigned for babies.”

CORE CONCEPT
The “Baby Core” must combine 4 normalized pillars:

1. Feeding
- milk intake / feeding frequency / solids appropriateness
- breastfeeding, bottle, pumped milk, or solids depending on age
- whether today’s feeding pattern is aligned with expectations for this baby profile

2. Sleep
- total sleep
- naps + nighttime sleep
- wake windows
- how well today’s sleep pattern aligns with age expectations

3. Hydration & Elimination
- hydration intake when relevant
- wet diapers
- dirty diapers / elimination pattern
- whether output seems normal for this baby’s context

4. Development & Regulation
- tummy time
- play
- reading / stimulation
- active engagement
- age-appropriate daily developmental activity and regulation

COMPOSITE SCORE LOGIC
The tile should visually communicate a single composite “Baby Core” result, but it must be clearly built from those 4 pillars.

The score should be adaptive and personalized based on:
- age in months
- sex, only when clinically relevant
- country / region / locale
- feeding type
- developmental stage
- climate / season if relevant
- prematurity or gestational adjustment if relevant
- other demographic context when useful

The product must communicate that the score is:
- contextual
- age-adjusted
- personalized
- a wellness/routine indicator
- NOT a medical diagnosis

MEDICAL / SAFETY TONE
Use medically cautious product language.
Do not imply diagnosis.
Do not present this as a medical device.
Frame it as:
- “routine wellness”
- “daily balance”
- “today’s core”
- “overall baby rhythm”
- “baby is on track today”
- “watch a few areas”
Avoid hard medical claims.

VISUAL DIRECTION
Create the tile in a premium light-mode baby-care aesthetic:
- soft off-white or warm light surface
- pastel accents
- subtle mint, baby blue, peach, lavender, soft rose
- large rounded corners
- clean spacing
- elegant health-product feel
- modern, soft, premium
- not childish
- not cartoonish
- not overly playful

VISUAL STRUCTURE
Design the tile as a large hero card for the top of the “Today” dashboard.

Suggested layout:
- left side: high-level summary text + 4 compact metrics
- right side: the visual “Baby Core” graphic

The core visual should feel inspired by Samsung Health’s core ring/heart style, but reimagined for babies:
- 4-part composite graphic
- smooth rounded geometry
- soft organic ring, petal, nest, clover, or layered shape
- each of the 4 pillars represented by its own color/segment
- one unified visual symbol showing balance and wellness
- should feel like a baby “health core”

Do NOT make it feel sporty.
Make it feel nurturing, balanced, and health-oriented.

CONTENT INSIDE THE TILE
Include:
- tile title, such as:
  - Baby Core
  - Daily Core
  - Baby Balance
  - Today’s Core
- a high-level status label, such as:
  - On Track
  - Doing Well
  - Balanced Today
  - Watch a Few Areas
- short supporting sentence
- 4 mini indicator rows or chips
- a single composite visual

EXAMPLE MINI INDICATORS
Use compact supporting stats such as:
- Feeding: On track
- Sleep: Good
- Diapers: Normal
- Activity: Slightly low

or:
- Feeding 86%
- Sleep 78%
- Hydration/Elimination 92%
- Development 70%

INTERACTION
Design both:
1. collapsed home tile state
2. expanded state / detail state

When expanded, the tile should reveal:
- contribution of each pillar
- short explanation of the score
- what is helping
- what may need attention
- trend vs yesterday or vs recent days

Create tap behavior concepts such as:
- tap tile → opens Baby Core details
- tap a pillar → opens detailed tracker view

DESIGN STATES
Create at least these tile states:
1. Excellent / very balanced day
2. Normal / on track
3. Watch / one or two weaker areas
4. Low-data state / not enough logs yet

TONE OF THE TILE
The tile should feel:
- premium
- reassuring
- insightful
- calm
- beautiful
- central to the product
- like the baby version of Samsung Health’s main daily core metric
