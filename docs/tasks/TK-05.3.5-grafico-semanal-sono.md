## Deliverable

Enhanced weekly sleep chart with gridlines, Y-axis labels (hours), background, auto-scaling, and interactive filter on the summary cards (sono noturno / cochilos).

## Acceptance Criteria

- Chart has subtle background and dashed horizontal gridlines
- Y-axis shows hour values with adaptive step (1h/2h/4h/6h)
- Chart auto-scales based on data range
- Summary cards ("sono noturno", "cochilos") are clickable to filter chart and history
- Active card highlighted with ring + scale; inactive card dimmed
- Clicking same card again resets to "all"
- Value labels shown above each bar when > 0
- History list filtered accordingly
