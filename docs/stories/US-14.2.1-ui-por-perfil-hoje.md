## User Story

As a caregiver, I want the Today home to reflect the enabled modules for my baby profile so that I only see trackers and sections that are rolled out for us.

## Acceptance Criteria

- Flags carregadas no bootstrap da app e expostas via contexto reutilizável
- `TodayPage`, `TodayTrackerGrid` e `TodayHealthSection` respeitam as chaves de flag acordadas
- Fallback seguro quando a API de flags falha (degradação controlada)
- Build do `app-design` conclui sem erros de tipo
