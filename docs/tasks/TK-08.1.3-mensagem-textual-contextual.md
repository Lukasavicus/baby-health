## Deliverable

Friendly, adaptive summary message that changes based on the day's data (e.g., "Great day! Davi slept well and ate plenty.").

## Acceptance Criteria

- Message tone is warm and non-judgmental
- No medical jargon or technical terms
- Message adapts to different metric combinations
- Multiple message templates to avoid repetition
