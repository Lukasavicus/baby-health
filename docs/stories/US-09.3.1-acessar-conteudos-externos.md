## User Story

Como cuidador, eu quero acessar conteúdos externos confiáveis para complementar as informações do app com fontes reconhecidas.

## Acceptance Criteria

- Dedicated external links section within content library
- Curated list of trusted sources only
- Tap on external link opens browser with exit warning
- Links organized by category
