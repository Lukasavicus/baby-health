## User Story

Como cuidador, eu quero uma área própria do bebê com dados centrais para visualizar rapidamente as informações mais importantes em um só lugar.

## Acceptance Criteria

- "Meu Bebê" tab accessible from bottom navigation
- Baby name displayed prominently
- Age calculated automatically from birth date
- Baby photo visible
- Associated caregivers listed
- Active trackers listed
- Quick access to sub-screens (growth, observations, etc.)
