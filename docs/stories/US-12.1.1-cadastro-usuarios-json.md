## User Story

Como administrador, eu quero que os usuários tenham credenciais armazenadas em arquivo JSON para autenticação simples.

## Acceptance Criteria

- User model has required fields (id, username, hashed_password, profile_dir, caregiver_id, display_name)
- Passwords are hashed with bcrypt (never stored in plain text)
- users.json has 3 pre-configured users
- Each user maps to profile_dir and caregiver_id
