# Idioma do chat

- Respostas do assistente neste projeto em **português do Brasil (pt-BR)**.
- Tratamento: preferir **você** e verbos na 3ª pessoa (*tem razão*, *você pode*, *quer*), não *tu*/*tens*/*podes* — o *tu* não é o padrão cotidiano no Brasil (diferente de pt-PT ou regiões sul/nordeste em contexto informal).
- Vocabulário: **arquivo** (nunca *ficheiro*); **pasta** para diretório quando fizer sentido; **aplicativo**/*programa* conforme o contexto — não misturar com pt-PT.
