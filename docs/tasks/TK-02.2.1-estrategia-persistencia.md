## Deliverable

Architecture decision record (ADR) for persistence strategy.
Options: Phase 1 (MVP) — JSON file (backend) + localStorage (frontend); Phase 2 — SQLite (backend) + IndexedDB (frontend); Phase 3 — Cloud DB (PostgreSQL) with local cache.
Decision: Phase 1 for MVP to minimize complexity; abstract API to allow swapping.

## Acceptance Criteria

- ADR document approved
