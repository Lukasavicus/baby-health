## User Story

Como cuidador, eu quero marcar se foi a primeira vez que um alimento foi oferecido para rastrear a introdução alimentar e prevenir alergias.

## Acceptance Criteria

- Checkbox: "First time"
- Can add food name when marked as first exposure
- First exposures tracked separately (for allergy prevention)
