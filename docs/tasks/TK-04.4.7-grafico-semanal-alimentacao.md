## Deliverable

Enhanced weekly feeding chart with gridlines, Y-axis labels, background, auto-scaling, and interactive filter icons in the summary (leite/sólidos/ml total).

## Acceptance Criteria

- Chart has subtle background and dashed horizontal gridlines
- Y-axis shows count/ml values with adaptive step
- Summary icons (leite, sólidos, ml total) are clickable to filter chart and history
- Active filter highlighted with ring + scale; inactive dimmed
- Clicking same filter again resets to "all"
- Custom food creation available when food search yields no results
