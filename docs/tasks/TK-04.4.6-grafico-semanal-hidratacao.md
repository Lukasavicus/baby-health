## Deliverable

Enhanced weekly hydration chart with gridlines, Y-axis labels, background, auto-scaling, and interactive filter icons in the summary section.

## Acceptance Criteria

- Chart has subtle background (`bg-secondary/20`) and dashed horizontal gridlines
- Y-axis shows ml values with adaptive step (50/100/200ml)
- Chart auto-scales based on data range
- Summary icons (ml total, água, outros) are clickable to filter chart and history list
- Active filter icon highlighted with ring + scale; inactive icons dimmed
- Clicking same filter again resets to "all"
- Quick-add buttons with descriptive text removed from summary section
- Value labels shown above each bar when > 0
