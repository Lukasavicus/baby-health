## User Story

Como cuidador, eu quero confirmar a administração de medicações para saber exatamente o que já foi dado ao bebê.

## Acceptance Criteria

- Logged medication marked as confirmed upon creation
- Alternatively, create as pending and check off later
- Timeline shows checkmark icon for confirmed doses
- Confirmation state persists across sessions
