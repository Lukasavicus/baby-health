## User Story

As a developer, I want a single hook for tracker event CRUD so that I don't repeat fetch/state/callback patterns in 8 pages.

## Acceptance Criteria

- useTrackerEvents hook encapsulates fetch, create, update, delete, refresh
- All 8 tracker pages use the hook
- No direct useState/useEffect for event management in pages
