## User Story

As a developer, I want shared UI components for charts, drawers, and log lists so that adding/maintaining trackers requires minimal boilerplate.

## Acceptance Criteria

- WeekBarChart used in all 7 tracker pages
- TrackerDrawer wraps all drawer instances
- EventLogList used for all tracker history lists
- No inline chart/drawer implementations remain
