## User Story

Como cuidador, eu quero um feed comum para acompanhar tudo que foi registrado por todos os cuidadores em um só lugar.

## Acceptance Criteria

- Caregivers tab shows a chronological feed of all events
- Feed displays the same events as the home timeline
- Each event shows caregiver info (name, avatar, color)
- Feed covers last 30 days by default
- Feed is refreshable (pull-to-refresh or refresh button)
