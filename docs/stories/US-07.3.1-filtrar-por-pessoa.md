## User Story

Como cuidador, eu quero filtrar registros por pessoa para ver apenas o que cada cuidador fez.

## Acceptance Criteria

- Filter chips showing each caregiver are visible at the top of the feed
- Tap on a chip to toggle that caregiver's filter on/off
- Timeline filters instantly when a chip is toggled
- "All" reset option returns the feed to showing all caregivers
- Filter state persists during the session
